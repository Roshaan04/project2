# Submission Form UI

This project is a submission form user interface created using HTML and CSS.
The layout is recreated from a reference design and focuses on achieving
sharp and clean edges using CSS clip-path instead of image rotation.

## Features
- Sharp diagonal background using CSS clip-path
- Clean and modern form layout
- Logo placed above the form
- Input fields, checkboxes, and text areas
- Proper alignment using position and z-index

## Technologies Used
- HTML5
- CSS3
- CSS clip-path

## Purpose
This project was created for practice to improve frontend development skills
and to learn modern layout techniques using pure HTML and CSS.
